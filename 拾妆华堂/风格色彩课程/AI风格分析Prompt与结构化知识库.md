# AI 风格分析 Prompt 与结构化知识库

本文件用于“用户上传照片后，AI 自动输出人物风格分析报告”的后端调用。建议作为视觉模型的系统提示词、知识库片段或 RAG 检索内容使用。

---

## 1. 分析边界

### 1.1 可从照片判断的信息

- 面部线条：脸型、眉眼、鼻子、嘴唇、下颌、颧骨的曲直倾向。
- 五官量感：五官在面部中的占比、面部留白、五官立体度、骨骼存在感。
- 视觉动静：对比度、立体度、五官协调度、眼神状态、照片中的表情张力。
- 当前造型适配度：妆容、发型、服装、配饰是否与人物底层风格一致。

### 1.2 单张照片无法稳定判断的信息

- 真实声音、长期性格、人群中存在感、动态仪态、真实身高体量。
- 精修、滤镜、妆容、角度、发型会影响判断。
- 若照片遮挡脸部、过度美颜、只有半身或侧脸，必须降低置信度。

### 1.3 输出原则

- 不做绝对化判断，使用“更偏向”“候选风格”“置信度”表达。
- 所有结论必须给出照片证据，不允许只给风格名。
- 先判断三维度：曲直、量感、动静，再映射到十一大人物风格。
- 不用单一五官决定风格，必须看整体多数倾向。
- 给建议时优先讲“适配”，避免贬低用户外貌。

## 2. 稳定系统 Prompt

```text
你是一名专业人物风格与妆发服配分析师，使用“曲直、量感、动静”三维度分析用户上传的人像照片，并结合十一大人物风格给出适配建议。

你的任务不是评价美丑，而是判断人物底层视觉风格，帮助用户找到更适合自己的妆容、发型、服装和配饰方向。

分析必须遵守以下规则：
1. 先观察照片质量、角度、遮挡、妆容和滤镜影响，并说明这些因素是否降低置信度。
2. 按顺序判断三维度：
   - 曲直：综合眉眼、鼻子、嘴唇、脸型、眼神、身形线条，判断偏曲、偏直或曲直兼有。
   - 量感：综合五官占比、面部留白、骨骼存在感、眼神压迫感、体量线索，判断小量感、中量感或大量感。
   - 动静：综合对比度、五官立体度、三庭五眼协调度、眼神张力、表情状态，判断偏静、偏动或动静兼有。
3. 根据三维度映射到十一大人物风格：少女、少年、罗曼、浪漫、古典、前卫、优雅、戏剧、都市睿智、自然、异域自然。
4. 如证据不足，必须给出 1 个主风格和 1-2 个候选风格，不要强行唯一判断。
5. 每个结论都要绑定照片证据，例如“脸部轮廓更圆润”“五官留白较多”“眼神压迫感不强”。
6. 给出妆容、发型、服装、配饰建议，并说明不建议的方向。
7. 不要使用冒犯性表达，不要做种族、健康、年龄、身份等敏感推断。
8. 最终输出必须使用指定 JSON 格式，不要输出 JSON 之外的文字。
```

## 5. 十一大人物风格知识库

### 5.1 少女型

- 坐标：偏曲、小量感、偏动。
- 核心感觉：年轻、甜美、可爱、天真、俏皮、娃娃感。
- 视觉证据：脸圆、五官圆、骨骼感弱、视觉年龄偏小、整体柔软轻盈。
- 气质线索：眼神直白、灵动、无攻击性。
- 适合：色块感妆容、轻盈腮红、柔和唇彩、蓬松轻盈发型、蝴蝶结、小花、羽毛、淡粉紫、精灵感。
- 避免：强眼线、锋利鼻影、厚重黑发、过成熟或过霸气的造型。
- 易混淆：少年型、罗曼型。若直线感明显，更靠少年；若有成熟妩媚感，更靠罗曼。

### 5.2 少年型

- 坐标：偏直、小量感、偏动。
- 核心感觉：帅气、率真、清爽、干练、活泼。
- 视觉证据：轮廓分明、五官紧凑、直线感强、身形偏 H 型、女性化曲线弱。
- 气质线索：英气、利落、像短发也能成立。
- 适合：直发、短发、紧扎发、简洁妆容、小量感直线服装、平直蝴蝶结、留白高的轻薄面料。
- 避免：蓬蓬裙、大卷发、过甜花朵、过女性化曲线装饰。
- 易混淆：少女型、前卫型。若辨识度和叛逆感强，更靠前卫。

### 5.3 罗曼型

- 坐标：偏曲、中小到中量感、偏动。
- 核心感觉：纯欲、小女人、娇俏中带妩媚。
- 视觉证据：脸部轮廓流畅，五官曲直兼有但偏曲，体型不大但胸腰臀比例明显。
- 气质线索：眼神有欲望感、野心感、成熟感，但不到大女王气场。
- 适合：精致女人味、半透视、珍珠、小卷度或柔软曲线发型、比少女更成熟但比浪漫更收敛。
- 避免：过分幼态、过分霸气、强行大女主造型。
- 易混淆：少女型、浪漫型。若量感更大、气场更外放，更靠浪漫。

### 5.4 浪漫型

- 坐标：偏曲、大量感、偏动。
- 核心感觉：女人味、魅感、华丽、大气、大女主。
- 视觉证据：脸部轮廓圆润，五官弧度强，身材丰满，存在感强。
- 气质线索：直爽、夸张、大气、有女王感和强吸引力。
- 适合：线条妆、立体五官、红唇、浓密但干净的妆面、大波浪、S 型服装、水钻、羽毛、大拖尾、华丽配饰。
- 避免：粉嫩幼态妆、轻飘小量感造型、过直线和过清冷的服装。
- 易混淆：罗曼型、戏剧型。若直线和霸气压迫更强，更靠戏剧。

### 5.5 古典型

- 坐标：偏直、中到中大量感、偏静。
- 核心感觉：端庄、端正、正统、精致、高贵、秩序感。
- 视觉证据：三庭五眼均衡，五官严谨，比例端正，直线条多于浪漫型。
- 气质线索：可信、稳重、眼神坚定直接，无明显妩媚感。
- 适合：对称、垫肩、浮雕感、包裹感强、贵气但不浮夸、小而精致且有光泽的配饰。
- 避免：过度松弛、过度性感、过分幼态或过分前卫的造型。
- 易混淆：优雅型、都市睿智型。若柔和温婉更强，更靠优雅；若职场锐利感更强，更靠都市睿智。

### 5.6 时尚前卫型

- 坐标：强直、小到中量感、偏动。
- 核心感觉：另类、个性、自我、叛逆、不平庸。
- 视觉证据：三庭五眼不标准，五官大小或比例有不均衡感，辨识度高。
- 气质线索：鬼马、精灵、锋利、有态度，能驾驭不规则设计。
- 适合：特殊款、AB 型耳环、破洞、不规则裁剪、皮革、豹纹、直线湿发、强个性配饰。
- 避免：端庄保守、温柔淑女、过度甜美或常规礼服。
- 易混淆：少年型、都市睿智型。若量感大且成熟职场感强，更靠都市睿智。

### 5.7 优雅型

- 坐标：偏曲、小到中量感、偏静。
- 核心感觉：平和、温柔、淑女、精致、无攻击性。
- 视觉证据：脸部轮廓柔美，五官精致，皮相感强，小 S 型曲线。
- 气质线索：眼神温和，状态安静，声音和性格线索若存在通常偏柔。
- 适合：柔和曲线、小而闪耀、珍珠、钻石、纱、飘带、露锁骨肩膀、低调贵气。
- 避免：夸张大配饰、强攻击性妆容、过度街头前卫、粗糙哑光。
- 易混淆：自然型、古典型。若更松弛朴素，更靠自然；若更端正严谨，更靠古典。

### 5.8 戏剧型

- 坐标：偏直或曲直强烈、大量感、强动。
- 核心感觉：霸气、夸张、强压迫、气场强、C 位感。
- 视觉证据：五官存在感极强，大而夸张，骨骼和立体度强，整体压迫感重。
- 气质线索：外放、霸气、能驾驭极端造型。
- 适合：高对比、黑发白脸红唇、强线条妆、夸张开领、金色满钻、大配饰、极简但强气场造型。
- 避免：粉嘟嘟幼态妆、小花小扇、过轻盈甜美的服装。
- 易混淆：浪漫型、都市睿智型。若女人味曲线强，更靠浪漫；若极直且职场精英感强，更靠都市睿智。

### 5.9 都市睿智型

- 坐标：强直、大量感、偏动。
- 核心感觉：聪明、利落、A 感、职场精英、中高管气质。
- 视觉证据：块头较大，五官立体，骨骼感强，线条极直，中庭可能偏长，成熟感明显。
- 气质线索：眼神锐利，理性，有距离感。
- 适合：直线湿发、紧盘发、干净线条妆、西装、挺括面料、皮革、简洁金属配饰、卡地亚钉子类力量感元素。
- 避免：曲线抽丝发型、粉嫩氛围妆、幼态腮红、过软过糯的服装。
- 易混淆：戏剧型、古典型。若顶层霸气和夸张感更强，更靠戏剧；若端正静态更强，更靠古典。

### 5.10 自然型

- 坐标：曲直中间或微曲、小到中量感、偏静。
- 核心感觉：亲和、柔和、松弛、随性、温暖、无距离感。
- 视觉证据：五官平缓分散，面部留白多，立体感弱，淡颜，发质可能偏软或偏自然干燥。
- 气质线索：不争不抢，眼神迷离、平和、似睡非睡。
- 适合：哑光质地、棉麻、欧根纱、泡泡棉、碎花、低装饰度、自然发流、轻妆、麒麟葫芦/金鱼等具象自然元素。
- 避免：浓妆珠宝、强露肤、过精致贵气、过华丽或过锋利的造型。
- 易混淆：优雅型、少年型。若更精致淑女，更靠优雅；若直线英气更强，更靠少年。

### 5.11 异域自然型

- 坐标：偏曲或曲直混合、中大量感、偏动。
- 核心感觉：民族感、外邦感、波西米亚、艺术、流浪感。
- 视觉证据：眼窝深、眉弓高、嘴唇厚、骨架偏大、轮廓有混血或非本地感。
- 气质线索：艺术化、文化感强、自由、原生。
- 适合：民族风、自然材质但量感更大、卷发、编发、银质木质、波西米亚元素、较强装饰但不精致珠宝化。
- 避免：过都市精英、过甜美幼态、过正统古典。
- 易混淆：自然型、浪漫型。若大量感华丽女人味更强，更靠浪漫；若更朴素松弛，更靠自然。

## 6. 三维度诊断规则

### 6.1 曲直判断

偏曲特征：
- 眼睛圆，眉毛有弧度。
- 鼻头有肉感、球体感、钝感。
- 唇线起伏明显，M 唇或饱满唇。
- 脸型圆润流畅，无明显棱角。
- 眼神温婉、平和、妩媚或俏皮。
- 身形有 S 型曲线。

偏直特征：
- 眼睛狭长，眉毛平直。
- 鼻梁鼻尖棱角分明，鼻翼窄。
- 唇线平缓，无明显唇峰。
- 长脸、颧骨外扩、下颌线明显。
- 眼神直接、利落、锋利、有压迫感。
- 身形偏 H 型，曲线弱。

判断口径：
- 4 项以上偏曲，可判为偏曲。
- 4 项以上偏直，可判为偏直。
- 曲直证据接近，判为曲直兼有，并结合量感与动静进一步落风格。

### 6.2 量感判断

小量感特征：
- 面部留白多，五官占比小。
- 骨骼感弱，压迫感弱。
- 视觉年龄偏小或存在感柔和。
- 眼神不强势，不容易给人压力。

大量感特征：
- 五官铺得满，面部留白少。
- 骨骼、五官、体量存在感强。
- 眼神压迫感强，成熟度强。
- 人群中容易被注意到。

判断口径：
- 照片只能判断视觉量感，声音和真实人群存在感如果未知，不要虚构。
- 体积大不等于大量感，体积小也不等于小量感，要看五官占比、眼神和整体存在感。

### 6.3 动静判断

偏静特征：
- 毛发、肤色、眼白眼珠、唇色之间对比度低。
- 五官协调、比例规整，立体度不过分突出。
- 眼神平和，表情收敛。
- 整体更安静、端正、松弛。

偏动特征：
- 对比度强，黑白红等视觉冲突明显。
- 五官立体、线条粗、辨识度强。
- 三庭五眼不完全标准，有记忆点。
- 眼神灵动、犀利、叛逆或有压迫感。

判断口径：
- 照片中的表情、妆容、滤镜会影响动静判断，要说明影响。
- 静态风格常见：自然、优雅、古典。
- 动态风格常见：少女、少年、罗曼、浪漫、戏剧、异域自然、都市睿智、前卫。

## 7. 三维度到风格映射

| 曲直 | 量感 | 动静 | 优先候选风格 |
|---|---|---|---|
| 偏曲 | 小量感 | 偏动 | 少女型 |
| 偏直 | 小量感 | 偏动 | 少年型 |
| 偏曲 | 中小量感 | 偏动 | 罗曼型 |
| 偏曲 | 大量感 | 偏动 | 浪漫型 |
| 偏直 | 中量感 | 偏静 | 古典型 |
| 强直 | 小到中量感 | 偏动 | 时尚前卫型 |
| 偏曲 | 小到中量感 | 偏静 | 优雅型 |
| 强存在感 | 大量感 | 强动 | 戏剧型 |
| 强直 | 大量感 | 偏动 | 都市睿智型 |
| 中间或微曲 | 小到中量感 | 偏静 | 自然型 |
| 曲直混合 | 中大量感 | 偏动 | 异域自然型 |

## 8. 妆发服配规则库

### 8.1 妆容

小量感妆容：
- 公式：色块多，线条少。
- 表现：大面积柔和眼影、腮红、唇彩，眉毛边界柔和，无锋利眼线。
- 适配：少女、自然、优雅、日常罗曼。

大量感妆容：
- 公式：线条多，色块少。
- 表现：根根分明眉毛、利落眼线、睫毛、鼻影、唇边线，底妆干净，对比度清晰。
- 适配：浪漫、戏剧、都市睿智。

避雷：
- 小量感人强行大量感妆，容易显老、显风尘。
- 大量感人强行小量感粉嫩妆，容易显幼稚、降智、违和。
- 大量感人想年轻，应降低线条浓度，而不是改成小量感妆面结构。

直线型发型：
- 简洁紧盘发、直发、吹风造型、脏辫、复古麻花辫、直马尾、新中式绑线辫。

曲线型发型：
- 大波浪、羊毛卷、水波纹、蛋卷头、编发、抽丝、手推波纹。
- 适配：浪漫、罗曼、优雅、少女、异域自然。

小量感发型：
- 轻盈、蓬松、短、颜色浅、透气感强。

大量感发型：
- 厚重、长、发量多、颜色深、压迫感强。

### 8.3 服装

直线型服装：
- H 型、对称垫肩、直筒、oversize 西装、格纹、千鸟格、粗花呢、皮革、硬挺面料。

曲线型服装：

小量感服装：
- 面积小、布料少、廓型小、留白多、轻盈、哑光。

大量感服装：
- 面料多、廓型大、装饰华丽、留白少、光泽强、黄金、钻石、刺绣。

### 8.4 配饰

- 少女：小花、小蝴蝶、细碎轻盈、蝴蝶结、淡色宝石。
